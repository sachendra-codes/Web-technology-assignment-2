3. What are XML parsers? Explain the types of parsers with their advantages and disadvantages.

An XML parser is a software library or package that provides interfaces for client applications to work with an XML document. The XML Parser is designed to read the XML and create a way for programs to use XML.

Advantages It supports both read and write operations and the API is very simple to use.
It is preferred when random access to widely separated parts of a document is required.
Disadvantages It is memory inefficient. (consumes more memory because the whole XML document needs to loaded into memory).
It is comparatively slower than other parsers.
